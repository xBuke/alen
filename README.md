# Orguljarstvo Kvaternik — Cursor paket v2

Ova verzija koristi provjerene podatke i fotografije s postojeće službene web-stranice.

## Najvažnije datoteke

- `ALEN_MEDIA_RESEARCH.md` — što je pronađeno i što nije potvrđeno
- `OFFICIAL_ASSET_MANIFEST.md` — sve fotografije sa službene stranice
- `CONTENT_VERIFIED.md` — potvrđeni tekstovi koji imaju prednost
- `siteContent.ts` — gotovi podatci za Next.js projekt
- `CURSOR_VERIFIED_ADDENDUM.md` — dodatne upute za Cursor
- `.env.example`

## Važno

Slike postojeće stranice većinom su stock fotografije. One jesu službeno korišteni vizuali, ali nisu potvrđeni projekti Orguljarstva Kvaternik.

Nije pronađen potvrđen javni portret Alena niti pouzdano indeksiran intervju. Nemojte koristiti tuđu osobu niti izmišljati medijske objave.
